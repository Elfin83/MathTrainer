MathTrainer - простой тренажер умножения и деления. 
При нажатии кнопки "Начать" появляется пример, необходимо ввести ответ в текстовое поле и либо нажать клавишу "Enter", либо мышью нажать кнопку "Готово".
Если ответ верный, появляется соответствующее сообщение и предлагается новый пример.
Если ответ неверный, появится сообщение с верным ответом и предлагается новый пример.
При появлении нового примера запускается таймер, если заданное время истекло, ответ засчитывается как неверный, появляется сообщение с верным ответом и предлагается новый пример.
Каждый пример учитывается в статистике, которая отображается в нижней части окна. Статистика учитывает общее количество примеров и количество правильных (в процентах).
Есть возможность изменить настройки приложения:
1. Операции, которые используются в примерах: деление, умножение. Можно решать примеры с обеими операциями или оставить одну из них.
2. Таймер: есть возможность полностью его отключить либо изменить лимит времени (от 3 до 60 секунд). По умолчанию таймер включен, ограничение - 5 секунд.
3. Числа. Выдкляются те из них, которые будут использованы в примерах на деление и умножении. Например, если выбрано только число 2, то будут предложены примеры умножения числа 2 на все другие числа от 0 до 9. Необходимо будет решить примеры 2х9 и 2х3, однако 3х9 не будет, так как для такого примера необходимо в настройках выбрать число 3. Так же с делением.

За одну тренировку должны быть предложены все варианты примеров (в соответствие с выбранными настройками), однако те примеры, которые решены с ошибками, появляются чаще тех, которые решены правильно.
Это реализовано с помощью таблицы весов. Каждый пример по умолчанию имеет вес 1, в случае неправильного ответа вес увеличивается на единицу и генератор случайных числе, основываясь на этом весе будет выбирать пример чаще.
Если введен правильный ответ, вес примера уменьшается на 1 (только если это не делает значение веса меньше 1).

Первоначально вес каждого примера 1. При генерации примера, объект с типом IExercise добавляется в словарь <string, IExercise>, где в качестве ключа выступает строка примера, например "2х3". "3х2" считает другим примером и добавляется отдельно.
